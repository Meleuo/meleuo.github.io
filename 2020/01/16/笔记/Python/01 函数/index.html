Python 函数

2020-01-16
  • 笔记

什么是函数

函数是一个封装好的可能会被重复用到的方法&工具，需要用的时候直接调用即可

函数的返回值

返回值和输出是有根本区别的

函数的参数

位置参数

位置参数就是对号入做，有几个行参就对应多少个位参

关键字参数

形参=实参 的方式绑定实参和形参，这样就只需要记住形参的名字不用记住位置。

默认参数

在不传送实参的情况下，具有默认参数的形参会直接调用默认参数，如果给有默认参数的形参传入实参，那么调用实参

动态参数

*arg

*arg 接受所有普通的实参，以tuple的形式输出

**kwargs

*kwarg 接受所有关键字参数，以dict的形式输出

混合参数

顾名思义，就是把前面几个参数混在一起使用！

位参必须定义在关键字参数前面

位参对应的形参不能再定义为关键字参数

匿名函数

当一些函数很简单，只是用一个固定的表达式计算一个值的时候就可以使用匿名函数

匿名函数正常都是使用return 返回结果，也没有谁会在匿名函数中定个print()

map() 高阶函数

map(func,iter)

函数接受2个参数，一个func 表达式函数，一个可迭代序列（看作列表即可），他的功能就是把序列中的每个值代入表达式函数计算。然后使用 list()列表函数取值
