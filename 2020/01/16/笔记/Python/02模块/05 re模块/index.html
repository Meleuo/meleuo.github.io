RE进阶

正则表达式也是一种语法,

我们写的正则也是需要被编译后才能被Python解释器所理解运行, 当需要重复使用同一条规则时可以使用re.compile 预编译好正则表达式赋予一个变量,这样使用的时候直接拿过来用, 一定程度上能够节省代码运行的时间效率

进阶 说明
re.compile 预编译一条正则表达式

在python通常会用iter() 把列表等数据类型创建一个迭代器对象next()可以依次取出迭代器中的元素

这样做的好处是: 使用迭代器可以调用出列表中的所有元素, 并且列表里面的所有元素不需要在同一时间加载到内存中, 需要用的时候加载即可, 一定程度上节省服务器的内存提高空间效率

re模块中也有同样功能的方法,

进阶 说明
re.finditer 把正则表达式匹配结果的对象以一个迭代器的方式返回

findall 分组优先

在findall中的正则表达式分组匹配的结果将会被优先显示, 即使是空

取消findall分组优先

同样适用于 re.compile

search + 分组

search也遵循分组优先的原则, 不同于findall的是, search 需要使用.group() 触发,并且可以指定输出哪一个表达式分组匹配到的内容

split + 分组

当split中的正则有分组，那么就会显示分组中切割消失的值

flags方法

flags方法 说明
re.I 忽略大小写,完整写法:IGNORECASE
re.M 忽略换行符，多行模式，改变^跟$的行为，完整的写法：MULTILINE
re.S 让.字符可以匹配任意字符，包括换行符，完整的写法：DOTALL
re.L 表示特殊字符集 \w, \W, \b, \B, \s, \S 依赖于当前环境，不推荐使用，完整的写法：LOCALE
re.U 使用\w \W \s \S \d \D使用取决于unicode定义的字符属性。在python3中默认使用该flag，完整的写法：UNICODE
re.X 冗长模式，该模式下pattern字符串可以是多行的，忽略空白字符，并可以添加注释，完整的写法：VERBOSE
