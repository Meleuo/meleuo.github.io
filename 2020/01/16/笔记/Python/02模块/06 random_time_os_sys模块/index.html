time 时间模块

三种时间

在python种通常时间分为三种格式

时间戳时间：又称之为格林威治时间，是float数据格式，时间戳时间就是从时间戳的起点到现在的每毫秒

* 时间戳时间的起始点是：英国伦敦时间：1970y1m1d  0:0:0

    * 英国伦敦时间和中国相差8个时区所以时间戳在中国的起点是：1970y1m1d  8:0:0

结构化时间 ： 元组struct_time类型

格式化时间（字符串时间）：通过用户自定义格式把结构化时间转换成人类可读的时间

time模块输出时间 说明
time.time() 返回时间戳时间，输出格式为:1573807230.5175846
time.localtime() 返回结构化时间，输出格式为：struct_time
time.strftime('%Y-%m-%d') 通过自定义格式返回格式化时间

时间转换

时间转换的原理

时间戳时间 > 结构化时间 > 格式化时间

time模块时间转换 说明
time.localtime(1573807945) 把时间戳转换为结构化时间
time.strftime('%Y-%m-%d',struct_time) 把结构化时间通过自定义格式转换为格式化时间

格式化时间 > 结构化时间 > 时间戳时间

time模块 时间转换 说明
time.strptime('2013-01-04','%Y-%m-%d') 把字符串时间转换为格式化时间
time.mktime() 把格式化时间转换为时间戳

补充

time.strftime() 支持的格式

time.localtime(). 支持取出的格式

索引（Index） 属性（Attribute） 值（Values）
0 tm_year（年） 比如2011
1 tm_mon（月） 1 - 12
2 tm_mday（日） 1 - 31
3 tm_hour（时） 0 - 23
4 tm_min（分） 0 - 59
5 tm_sec（秒） 0 - 60
6 tm_wday（weekday） 0 - 6（0表示周一）
7 tm_yday（一年中的第几天） 1 - 366
8 tm_isdst（是否是夏令时） 默认为0

sys 解释器交互模块

sys模块 说明
sys.path 返回模块搜索路径，返回结果为list 初始路径是PTRHONPATH环境变量中定义的
sys.modules 返回当前文件运行时加载的模块，返回结果为dcit
sys.argv[n] 读命令行传递的参数，argv[0]对应的是当前.py文件本身（文件名）
sys.version 输出当前Python解释器版本
sys.exit 退出程序
sys.platform 返回操作系统类型，结果并不准确

os 系统交互模块

OS模块 就是python和系统交互的接口

文件目录操作

os模块目录文件操作 说明
os.mkdir('dir') 创建一个目录，已存在则报错
os.makedirs(’dir/dir2‘) 递归创建目录，隐藏参数: `exist_ok=Trur False(default)，exist_ok为False`时创建的最后一个目录如果存在则会报错。
os.rmdir(’dir‘) 删除一个空目录，必须是空否则报错
os.removedirs('dir/dir2') 递归删除一个空目录，所有被递归删除的目录下不能有文件
os.remove('file') 删除一个文件
os.rename('src','dst') 修改文件名
os.listdir('dir') 列出文件下所有文件和其子目录包括隐藏文件不递归，输出格式：list
os.stat('') 获取文件&目录详细信息

系统信息

OS模块系统信息 说明
os.sep 输出当前系统使用的目录分隔符，Windows为\，类uinx为/
os.linesep() 输出当前系统使用的行终止符，Windows为\r\n，类uinx为 \n
os.name 输出当前系统类型，Windows为 nt，类uinx为 posix

os.path.dirname(path) 把传递过来的路径规范化后，切割最右的一个路径然后和前面的拼成二元组返回，如果最右的是文件系统的分隔符/则第二个元素就是空

.dirname返回.split第一个元素

OS模块 os.path 说明
os.path.dirname(path) 返回split第一个元素

.basename返回.split第二个元素

OS模块 os.path 说明
os.path.basename(path) 返回split第而个元素

请注意.dirname和.basename不是真的拿.split的结果返回，而是返回的结果刚好和.split的2个元素一模一样，所有就这么叫，就不用做多余的解释
