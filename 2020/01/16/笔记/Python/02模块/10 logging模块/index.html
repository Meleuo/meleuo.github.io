logging只是用来保存或者打印你定义的报错信息。所有格式，打印的格式，内容都要你自己写

日志等级

日志等级 说明
logging.debug() debug 调试信息
logging.info() 正常普通的通知信息
logging.warning() 警告信息
logging.error() 错误信息
logging.critical() 严重错误

普通配置型

配置比较简单，但是无法实现既打印在屏幕上面又同时往文件里记录，也无法解决文件名是中文时记录的日志中存在乱码的情况

logging.basicConfig 参数 说明
level = 定义日志最低显示级别
format = 定义日志输出格式
darefmt = 定义日志中的时间格式
filename = 输出到指定文件
