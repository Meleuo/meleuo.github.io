subprocess 模块

在os模块中有一个popen模块可以执行操作系统的命令，但是！popen方法只能返回命令执行正确的结果，如果命令执行出错返回来的结果就是乱码，而且这个乱码是无解的。

subprocess.Popen() 就是用来弥补这个缺陷的，虽然使用相对 os.popen()要麻烦一点，但是好在人家能够正常使用

subprocess.Popen() 方法参数 说明
`shell=Ture False(default)` 表示是否把cmd当初系统名字执行
stdout=subprocess.PIPE 指定正确输出后存放缓冲区管道[^1注释]
stderr=subprocess.PIPE 指定错误输出后存放缓冲区管道

[^1注释]: 缓存区的数据只能被 .read()一次，读一次就没了，常用的手段就是读取来，赋予给变量

struct模块

struct模块用于把一串数字[^注释]转换为一个 4字节bytes

[^注释]: 数字的长度受python 中最大数字的现在，正整数是2147483647，负整数是-2147483648，可以使用 sys.maxsize 查看

struct 常用方法 说明
struct.pack() 把数字压缩成 4个bytes类型
struct.unpack() 解压，结果以一个元组返回
