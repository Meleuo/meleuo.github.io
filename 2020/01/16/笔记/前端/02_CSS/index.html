display参数 说明
none 元素存在在代码上，但是不在浏览器上显示，一般配合js使用
block 让行内元素拥有块级元素的特性，沾满页面宽度，可设置宽高，

浮动

清除浮动

解决父标签塌陷问题，如下，使用浮动的标签就脱离了原来的HTML结构

解决方法：

  • clear属性
  • 给父标签设置固定高度
  • 伪元素
  • overflow:hidden

clear属性在标签最下面添加一个拥有clear属性的div空标签

给父标签设置固定高度

伪元素

借助伪元素 after 在标签最后面实现一个空的块级区域， 然后借助clear属性避免塌陷

溢出

内容溢出后使用overflow 控制溢出内容的效果

overflow参数 说明
visible 默认值，溢出内容不会被处理，会呈现在元素框外
hidden 溢出内容会被隐藏，不可见
scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。
inherit 从父元素继承

定位

定位position属性是和 left, right, top, bottom这样的位置属性是一起用的， 简单来说position是定义位置属性的参照物

position参数 说明
static 默认， 无定位，不能当中绝对定位的参照物,并且设置位置属性（left,top等）是不起作用的
relative 相对定位， 先对于元素原始位置进行定位
absolute 绝对定位，参考上级已经定位的元素进行定位，如果没有上级元素，或者上级元素没有定位那么将会依照body标签定位（页面左上角）
fixed 固定定位，参考浏览器页面窗口进行定位

绝对定位， 固定定位都和浮动一样脱离原有的页面结构，相对定位不会脱离

z-index

设置元素在网页上面的层叠顺序，并且元素必须设置定位position才可以设置z-index

opacity

设置元素透明度
