方法 说明

.splice()

.splice(start, end, str1,str2…)

start 和end是必填表示删除的起点和终点， str1.替换被删除数据

.forEach()

.forEach(function (a,b,c){}) 中定义一个匿名函数，forEach方法会将list的值循环传递给匿名函数，匿名函数可以接受3个参数，a是循环传递过来的值， b是a在list中的下标， c是整个list

.map()

.map()接.forEach()的区别是map会将匿名函数处理的结果返回回来，可以通过赋值的方法得到经过匿名函数处理后的list

BOM和DOM

BOM

BOM 开放给用户操作浏览器的方法

window

windows 所有浏览器都支持的对象, 他表示的就是浏览器窗口

所有 JavaScript 全局对象、函数以及变量均自动成为 window 对象的成员

全局变量是 window 对象的属性。全局函数是 window 对象的方法。

常用的事件触发

代码 事件
onclick 鼠标单击某个对象事件
ondblclick 鼠标双击某个对象事件
onfocus 元素获得焦点, 常用于input
onblur 元素失去焦点, 常用于input
onchange 域的内容改变, 通常用户select的option的变化
onkeydown 某个键盘按键被按下。 应用场景: 当用户在最后一个输入框按下回车按键时,表单提交
onkeypress 某个键盘按键被按下并松开
onkeyup 某个键盘按键被松开。
onload 一张页面或一幅图像完成加载。
onmousedown 鼠标按钮被按下
onmousemove 鼠标被移动
onmouseout 鼠标从某元素移开
onmouseover 鼠标移到某元素之上
onselect 在文本框中的文本被选中时发生
onsubmit 确认按钮被点击，使用的对象是form
