ubuntu安装N卡驱动

2020-07-01
  • Linux

!!!本文只适用于使用了默认nouvea驱动, 且使用独显的ubuntu机器


Ubuntu系统集成的显卡驱动程序是nouvea，

所以需要先将nouveau驱动从Linux内核中屏蔽掉

才能安装NVIDIA官方驱动。



屏蔽nouveau驱动


然后重启操作系统

需要注意由于禁用了nouveau驱动再次开机会进不去desktop， 需要使用 Ctrl + Alt + F2 进入终端模式，

所以建议先下载好驱动文件再重启



安装N卡驱动

在这里寻找您需要掉驱动：https://www.nvidia.cn/Download/index.aspx?lang=cn

我这里用的显卡是1050TI， 安装的版本是450

https://www.nvidia.cn/Download/driverResults.aspx/165210/cn

NVIDIA-Linux-x86_64-450.80.02.run



安装好了后就会自动进入桌面， 推荐使用测试
