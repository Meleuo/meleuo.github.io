基于MySQL innodb cluster 的MySQL高可用方案

2020-10-12
  • Mysql

MySQL InnoDB Cluster是MySQL 推出的最新的高可用方案，

  • 每个实例都运行 MySQL Group Replication, 它提供了具有内置故障转移功能的innodb cluster，

  • 借助MySQL shell 可以轻松管理cluster， 相比较传统的主从复制方案配置更见方便， 状态清晰明确。

  • MySQL router 提供了路由功能，使用他你可以不需要关心集群中那个是主节点， 当主节点出现故障时， router也会自动帮你路由到新的主节点上。

Router高可用 keepalived

keepalived高可用是给 MySQL Router准备的, MySQL集群已经实现了高可用,但是 如果入口Mysql router挂掉了集群还是等于挂逼了, 这里就使用keepalived + VIP的机制给Mysql router实现高可用


借助 keepalive + VIP , master mysqlrouter 挂掉可以立马切换到BACKUP, 配置notify对管理员进行通知
